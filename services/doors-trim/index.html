Doors & trim FAQs

Can you match existing trim profiles?

We stock common profiles and can source close matches. We’ll discuss options if an exact match isn’t available.

Do you replace door hardware?

Yes, we install new hinges, locks, and handles and align them for smooth latching.

Can you work in occupied suites?

Absolutely. We use drop cloths, HEPA vacuums, and tidy up before we leave.
